                  南京理工大学成立于 1953 年，由哈尔滨军事工程学院
              ( 哈军工 ) 分建而成，先后经历炮兵工程学院、华东工程
              学院等几个发展阶段，1993 年更为现名。其兵器博物馆珍
              藏了自第一次世界大战以来，世界多个国家不同历史时期
              的火炮类、轻武器类、导弹类等诸多门类的现代兵器装备，
              以数量众多、专业性强闻名。其中许多为我国自行研制的
              武器装备。南京理工大学兵器博物馆平均每天需接待 3 场
              左右的参观团，工作量繁杂，而铸剑博闻社兵博精讲团则
              是协助兵器博物馆完成日常接待任务的中坚力量。
